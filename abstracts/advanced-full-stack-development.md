# Advanced Full-Stack-Development: How to use React and React Native to build a unified web and mobile experience

Full-Stack-Development of web apps can be difficult, but what if you extend Full-Stack to include mobile apps as well? How can you handle this additional dimension of complexity as efficient as possible?

In this talk, you will learn how you can create a React web client and a React Native mobile client for a Spring Boot backend in a very productive way. We will explore concepts for code generation, reusable libraries and components and integration techniques to write less code and to be able to run this code almost everywhere. Expect a lot of code examples and practical tips and tricks.
